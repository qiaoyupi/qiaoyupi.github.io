    位置错误

    常规方法，循环内调用接口，会出现放回的值所在数组的位置错误

    碰到这种情况如下解决

    this.apiList = []
this.ids = []
this.list = []
ids.forEach((el) => {
  apiList.push(new Promise((resolve, reject) => {
    Api.info(el).then((res) => {
      return resolve(res)
    }).catch(err => {
      return reject(err)
    })
  }))
})

Promise.all(apiList).then(res => {
  this.list = res
}).catch(() => {
  this.list = []
})

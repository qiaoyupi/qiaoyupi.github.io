# 添加到数组，接口返回值的顺序不对

解析

因为接口返回的时间长短不一，调用的最晚的接口不一定最后响应回来。所以出现了返回值在数组中与接口的位置不一致的问题

# 解决方案













 





const apiList = []
let list = []
[].forEach((el) => {
  apiList.push(new Promise((resolve, reject) => {
    Api.info(el).then((res) => {
      return resolve(res)
    }).catch(err => {
      return reject(err)
    })
  }))
})

Promise.all(apiList).then(res => {
  list = res
}).catch(() => {
  list = []
})

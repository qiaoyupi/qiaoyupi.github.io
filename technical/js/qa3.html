ES6的Set和Map

Joe 22/6/21 15:47 JS ES6

# 1. Map

  • Map 是一组键值对的结构，具有极快的查找速度
    • 存有键值对，其中键可以是任何数据结构
    • 记得键的原始插入顺序
    • 具有表示映射大小的属性
  • 与对象的差异 {}
    • 对象没有size属性
    • 对象的键必须是字符串 （或符号）
    • 对象没有很好的排序
    • 对象有默认键

# 可以传递一个数组

const m = new Map([['a', 1], ['b', '2'], ['c', 'cc']]) // {'a' => 1, 'b' => '2', 'c' => 'cc'}

# 2. Set

  • Set是唯一值的集合
    • 存key，并且不重复
    • 可以容纳任何数据类型的任何值
    • 没有顺序概念
  • 对数组的差异 []
    • 元素不唯一
    • NaN是不同元素 // 在Set里面NaN是一个元素

# 3. 拓展 并集、交集、差集

概念

并集的数学概念，集合A和集合B的并集，表示为A∪B
交集的数学概念，集合A和集合B的交集，表示为A∩B
差集的数学概念，所有属于A且不属于B的元素构成的集合，叫做集合A减集合B，表示为A-B；对于集合A、B，把集合{x∣x∈A，且x∉B}叫做A与B的差集
子集的数学概念，集合A的任意一个元素都是集合B的元素，表示为A⊊B

# 并集

Set.prototype.union = function (val) {
  const union = new Set()
  this.forEach((el) => {
    union.add(el)
  })
  val.forEach((elm) => {
    union.add(elm)
  })
  return union
}

new Set([1, 2, 3, 4]).union(new Set([3, 4, 5, 6]))
// 1...6

# 交集

Set.prototype.intersection = function (val) {
  const intersection = new Set()
  if(this.size > val.size) {
    val.forEach((el) => {
      if (this.has(el)) {
        intersection.add(el)
      }
    })
  } else {
    this.forEach((el) => {
      if (val.has(el)) {
        intersection.add(el)
      }
    })
  }
  return intersection
}

new Set([1, 2, 3, 4]).intersection(new Set([3, 4, 5, 6]))

# 差集

Set.prototype.difference = function (val) {
  const difference = new Set()
  this.forEach((el) => {
    if (!val.has(el)) {
      difference.add(el)
    }
  })
  return difference
}

new Set([1, 2, 3, 4]).difference(new Set([3, 4, 5, 6]))
// 1, 2

# 子集

// A 是 B 的子集
Set.prototype.subset = function (val /* B */) {
  if (this.size > val.size) {
    return false
  }

  let f = true
  this.forEach((el) => {
    if (f && !val.has(el)) {
      f = false
    }
  })
  return f
}

new Set([1, 2, 3, 4]).subset(new Set([3, 4, 5, 6]))
// false
new Set([3, 4]).subset(new Set([3, 4, 5, 6]))

# 5. Array.from

# 第三个参数 this指向

我们可以将处理的数据和处理的对象分离。将各种不用的数据处理方法封装到不同的对象中去，处理方法采用相同的名字
在调用Array.from对数据对象进行转换时，可以将不同的处理对象按实际情况进行注入，以得到不同的结果，适合解耦
这种做法是模版设计模式的应用，有点类似于依赖注入

const obj = {
  handle (n) {
    return n*=2
  }
}

Array.from([0, 1,2,3,5,6], function (n) {
  return this.handle(n)
}, obj)

// (6) [0, 2, 4, 6, 10, 12]

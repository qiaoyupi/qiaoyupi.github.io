伪类

Joe 22/6/16 CSS

伪类一般用一个冒号 :first-child, 伪元素用两个冒号 ::after，大多数浏览器也是支持一个冒号的伪元素

# :focus-within

<div class="box">
   <label for="label">提示</label>
   <input id="label">
</div>

问题

input 处于focus状态时，只能改变自身的样式。如果需要改变父级或者同级的样式，怎么做？

解析

HTML 提供了一个伪类，:focus-within。 这个伪类在自己或后代元素获取焦点时触发。

  1. 具有冒泡性质，后代获取焦点，层层递增，每一级都可以触发:focuus-within
  2. 感知获焦区域变大，同样改变感知获焦区域的，还有 label标签


 













.box {
   position: relative
   &:focus-within {
      label {
         top: -20px;
      }
      input {}
   }
   label {
      position: absolute;
      top: 0;
   }
   input {}
}
